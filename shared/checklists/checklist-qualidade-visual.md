# Checklist de Qualidade Visual

**ID:** MKTG-CL-001
**Usado por:** Nina (aprovacao-outputs), Diego (antes de entregar)
**Propósito:** Validar peças visuais (carrosséis, stories, posts) antes da publicação.

## 1. Identidade Visual
- [ ] Design System correto para a marca? (CRITICAL)
- [ ] Cores dentro da paleta? (CRITICAL)
- [ ] Tipografia correta (display + corpo)? (CRITICAL)
- [ ] Logo presente no rodapé?
- [ ] Elementos gráficos da marca (diagonal/ondas/gradientes)?

## 2. Hierarquia e Composição
- [ ] Palavra-chave em destaque na cor da marca? (CRITICAL)
- [ ] Hierarquia visual clara (título > subtítulo > corpo > CTA)?
- [ ] Espaçamento generoso (mín 40px margens)?
- [ ] Alinhamento consistente (nada "no olho")?
- [ ] Contraste suficiente entre texto e fundo?

## 3. Legibilidade Mobile
- [ ] Texto mínimo 14px equivalente? (CRITICAL)
- [ ] Legível em tela de celular?
- [ ] Numeração de slides consistente (NN/NN)?
- [ ] CTA visível e claro?

## 4. Técnico
- [ ] Formato correto (1080x1350, 1080x1920, 1080x1080)? (CRITICAL)
- [ ] Imagens sem artefatos, mãos deformadas ou baixa resolução?
- [ ] Fontes renderizadas corretamente?
- [ ] PNG com menos de 2MB?

## 5. Coerência
- [ ] Tom visual alinhado com o Brand Guide da marca?
- [ ] Não mistura elementos de outras marcas?
- [ ] Capa dark/forte + conteúdo alternado + CTA na cor da marca?
- [ ] Isso faz o olho brilhar? (CRITICAL)

## Veredicto
- **APPROVE:** Todos CRITICAL ok + menos de 2 falhas
- **REVISE:** Todos CRITICAL ok + 2 ou mais falhas
- **REJECT:** Qualquer CRITICAL falhou
